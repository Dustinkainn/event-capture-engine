# Build Roadmap

## Phase 1: Core Capture

Goal: prove the system can capture clean event data.

Build:

- Event builder
- Public registration page
- Attendee and group registration
- Basic custom questions
- Local database records
- Count summaries

## Phase 2: Check-In

Goal: prove the event-day workflow works.

Build:

- QR generation
- Confirmation email
- Scanner page
- Manual search
- Check-in logs
- Attendance counts

## Phase 3: MinistryPlatform Sync

Goal: prove clean event data can move to MinistryPlatform reliably.

Build:

- MinistryPlatform connector
- Contact matching
- Registration sync
- Attendance sync
- Sync logs
- Failed-sync review screen

## Phase 4: Payments and Polish

Goal: support paid events without breaking counts.

Build:

- Stripe checkout
- Ticket types
- Promo codes
- Manual comps
- Payment status
- Refund tracking
- Better reports
