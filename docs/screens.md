# Screens

## Admin Dashboard

Staff control center for upcoming events, event status, registered count, checked-in count, payment status, failed syncs, count summaries, attendee search, CSV export, print roster, and reconciliation.

## Event Builder

Create and manage internal events.

Fields:

- Event name
- Date and time
- Location
- Description
- Capacity
- Registration open and close dates
- Event image
- Free or paid event toggle
- Public or private event page
- Draft, open, closed, archived status
- Internal notes

## Registration Form Builder

Configure contact fields, attendee fields, group or family registration, custom questions, required fields, conditional questions, waivers, consent checkboxes, and count-mapped answers.

Question types:

- Text
- Email
- Phone
- Dropdown
- Multiple choice
- Checkbox
- Date
- Long answer
- Waiver or consent

## Public Registration Page

Registrant-facing event page for signup, attendee entry, custom questions, waivers, and payment handoff when needed.

## Confirmation Page

Shows successful registration details, ticket or QR access, resend option, and event-specific next steps.

## Check-In

Event-day interface for QR scanning, manual attendee search, group check-in, individual check-in, undo check-in, already-checked-in warnings, and check-in logs.

## Count Summary

Operational count view for total registrations, total attendees, meals, shirt sizes, childcare, sessions, volunteers, transportation, custom categories, payment-based counts, and checked-in totals.

## Reconciliation

Staff review screen for comparing registrations, attendees, payments, check-ins, counts, and sync readiness before final reporting or MinistryPlatform sync.

## Sync Review

Manual review screen for MinistryPlatform matching, queued syncs, retries, failed syncs, payload snapshots, and audit history.
